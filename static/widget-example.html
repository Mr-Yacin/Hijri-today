Complete Integration Code

Copy and paste this complete code to integrate the widget into your website:

<!-- Basic Integration --> <div data-hijri-widget></div> <script src="https://your-domain.com/hijri-widget.js"></script> <!-- With External CSS (Recommended) --> <link rel="stylesheet" href="https://your-domain.com/hijri-widget.css"> <div data-hijri-widget data-lang="ar" data-theme="dark"></div> <script src="https://your-domain.com/hijri-widget.js"></script> <!-- Multiple Widgets --> <div data-hijri-widget data-country="SA" data-lang="en"></div> <div data-hijri-widget data-country="MA" data-lang="ar"></div> <div data-hijri-widget data-country="TR" data-theme="compact"></div> <script src="https://your-domain.com/hijri-widget.js"></script>

JavaScript API Examples

You can control widgets programmatically using the JavaScript API:

// Refresh all widgets HijriWidget.refresh(); // Get widget statistics const stats = HijriWidget.getStats(); console.log('Total widgets:', stats.totalWidgets); // Get specific widget instance const widget = HijriWidget.getInstance('my-widget-id'); if (widget) { // Update widget configuration widget.updateConfig({ lang: 'ar', country: 'EG', theme: 'dark' }); // Refresh specific widget widget.refresh(); }
